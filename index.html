02 理论提出背景

1
资本主义空间组织转型

70年代后福特主义转向，资本积累从“国家内向型”走向“全球流动型”，传统国家“容器”功能失效。

2
传统国家理论的「空间缺失」

经典理论偏重权力与阶级，忽视空间。无法解释为何国家会“策略性地”倾斜扶持特定区域。

3
现实治理危机：去国家化 vs 再国家化

全球化带来权力外溢的同时，国家正通过重塑空间夺回竞争力。这一悖论催生了新解释框架。

03 理论框架与本体论

Ontology / 本体论

国家空间不是给定的地理容器，而是社会建构的过程，是资本、权力和斗争不断重组的结果。

Mechanism / 核心机制

  • 1. 国家空间选择性 (Selectivity) 资源分配的偏袒性与目的性。
  • 2. 尺度重组 (Rescaling) 权力在不同空间层级的位移表现。

Analysis Tool / TPSN 框架

T Territory 领土
P Power 权力
S Scale 尺度
N Network 网络

04 核心术语深度释义

国家空间选择性 (Spatial Selectivity)

核心机制

国家并非一视同仁，而是有策略地配置权力与政策。本质是国家在空间上的偏袒性干预，是理论运行的灵魂。

空间策略 (Strategies)

顶层逻辑与导向。决定了“为什么要这么规划”。例如：参与全球竞争。

空间形式 (Forms)

制度硬件。包括行政区划、国土规划体系、特殊管理体制。

空间项目 (Projects)

落地抓手。具体的工程与扶持政策，如特区建设、大型基建。

06 典型应用案例

案例 A：中国改革开放以来的空间重构 ▼

空间项目：深圳特区、浦东开发、雄安新区、自贸区。

选择性表现：国家资源向沿海、核心城市群、特定国家战略区域的梯度倾斜。

尺度重组：权力下放至省市层级，同时加强超国家（一带一路）层级的对话。

案例 B：欧盟一体化与跨境治理 ▼

典型的“权力上移”案例。通过设立欧盟共同政策（CAP、ERDF）实现跨国界的尺度重塑与资源选择性扶持。

理论贡献 / Contribution

  • ✓ 填补国家理论「空间空白」，证明国家转型即空间转型。
  • ✓ 超越了简单的“国家消亡论”，揭示国家如何主动重构空间。
  • ✓ 提供了极具操作性的分析工具（尺度、选择性、TPSN）。

局限与发展 / Limits

  • ! 西方中心主义倾向：对非西方国家的特殊治理逻辑讨论仍有待加强。
  • ! 侧重宏观结构：对微观社会力量、生态文化维度的关注相对偏弱。

极简总结

新国家空间理论通过空间选择性这一核心机制，揭示了国家在全球化浪潮中不再是静止的容器，而是一个通过不断进行尺度重组、主动制造地理不平衡以换取竞争力的策略性主体。
